The taxonomy

Fifteen domains, an expanding taxonomy of verbs

Assessing each domain the same way surfaces a one-word verb — the dominant shape of its gap. A diagnostic that classifies any NYC domain in one pass. The first five domains yielded five verbs; the next ten expanded the set — and clustered around one theme.

The evidence

Maturity scorecard

Each domain scored 0–3 across seven dimensions. Hover a cell for the reasoning. Two rows are effectively all zeros — the citywide gaps.

The universal gap

Every domain has one un-API'd citizen action

Open data liberated reporting; it never touched transactions. Fourteen of fifteen domains have a citizen action with no API — the highest-value net-new work. (The fifteenth, City Planning, is a reference agency with no citizen transaction.)

The latent model

A shared geography spine already recurs everywhere

Writing one _common.json per domain surfaced the same fields over and over — a citywide data model hiding in plain sight, and the key to cross-domain interoperability.
