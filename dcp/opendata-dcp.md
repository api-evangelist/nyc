# NYC Open Data — Department of City Planning (DCP)

Every NYC Open Data asset published by **Department of City Planning (DCP)** — the exact `Dataset-Information_Agency` label (verified 2026-07-13 via the Socrata Discovery API against `data.cityofnewyork.us`). Machine-readable: [opendata-dcp.json](opendata-dcp.json). Companion: [crosswalk.md](crosswalk.md).

**188 assets** — by type: dataset (102), map (61), file (20), href (5).

DCP is unusual: it is the **source of the city's shared geography** — the BBL (via PLUTO/MapPLUTO), community districts, Neighborhood Tabulation Areas (NTAs), census tracts, council/election district boundaries, and borough boundaries that every other agency's data references. Its Open Data footprint is geography and demographics, not transactions.
