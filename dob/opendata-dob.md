# NYC Open Data — Department of Buildings (DOB)

All **44** datasets on NYC Open Data whose **Dataset-Information_Agency = "Department of Buildings (DOB)"** (verified via the Socrata catalog API, 2026-07-13), sorted by lifetime page views. Machine-readable index: [opendata-dob.json](opendata-dob.json).

> Every one of these is a **nightly batch extract** of the BIS / DOB NOW application layer — the recurring `DOBRunDate` column is the tell. Open Data is the *only* machine-readable DOB surface, and it is one-way and up to a day stale.
