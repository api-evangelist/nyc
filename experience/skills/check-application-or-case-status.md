# Check application or case status

*Common government process — Know · Read · status.* Part of the [NYC Programmable City](https://nyc.apievangelist.com/experience.html) experience layer.

**Look up the status of an application, permit, complaint, case, or request across agencies.**

## When to use

The resident wants to know where something stands. Identify the case by its id/owner and return its current status and history.

## What it orchestrates

This skill spans **23 agencies** and **23 operations** across the corpus. It resolves the right agency for the task, then calls that agency's MCP tools (each backed by a REST operation) to complete it — anchoring any location on the [nyc-commons](https://nyc.apievangelist.com/commons.html) geography spine.
