# Request public records

*Common government process — Act · Request-records.* Part of the [NYC Programmable City](https://nyc.apievangelist.com/experience.html) experience layer.

**Make a records request — a FOIL request, a vital record (birth/death/marriage), a certified document, or a transcript.**

## When to use

The resident needs an official copy or a records disclosure. Identify the record type and custodian agency, submit the request, and track fulfillment.

## What it orchestrates

This skill spans **2 agencies** and **2 operations** across the corpus. It resolves the right agency for the task, then calls that agency's MCP tools (each backed by a REST operation) to complete it — anchoring any location on the [nyc-commons](https://nyc.apievangelist.com/commons.html) geography spine.
